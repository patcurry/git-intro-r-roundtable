What is Git?

Git is a version control system.

It is a way to track changes in computer files.

Git is also a way for multiple people to work on the same file or files at the same time.

Git is not easy, even though every tutorial out there says that it is.

Creating a repository

First local, then remote

Go to your documents directory then create a new folder initialize a git repository in that folder.

              
                cd ~/Documents
                mkdir learngit
                cd learngit
                mkdir firstrepo
                cd firstrepo
                git init

Now that you've created the repository let's add a file.

              
                pwd
                touch test.txt

Let's set a remote repository.

              
                git status
                git remote add origin https://github.com/USERNAME/learngit

Set git username and email

            
              git config --local user.name "USERNAME"
              git config --local user.email "EMAIL"

Push initial commit to the remote repository

              
                git push origin master

It didn't work!!!!

Cloning a repository

Go to repo you want to clone and find the 'clone or download' button.

Select 'Clone with HTTPS' and copy the URL

Open your terminal and find a good spot for a new folder, and clone the repository.

              
                git clone https://github.com/patcurry/rroundtable-git-clone-practice.git

Now you have a local version of the repository.

Let's take a look at the commit log

              
                git log

Let make this easier to look at

              
                git log --oneline --all

Graphical

              
                git log --oneline --all --graph

More info

              
                git log --oneline --all --graph --decorate

Even More info

              
                git log --all --graph --decorate
